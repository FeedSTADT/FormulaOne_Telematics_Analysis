This project presents a Formula One telemetry simulation and analysis framework developed using Python and Machine Learning techniques. The objective was to generate realistic synthetic telemetry data that replicates the behavior of an F1 car under varying race conditions while incorporating insights derived from real-world Formula One datasets. The simulation includes key telemetry parameters such as speed, engine RPM, brake pressure, steering angle, throttle input, lap performance metrics, and vehicle dynamics data, creating a structured dataset suitable for analytical and predictive applications.

The synthetic telemetry data was processed using Pandas and analyzed through a series of visualizations designed to capture vehicle behavior, braking characteristics, speed fluctuations, and driver control patterns. To improve the realism of the generated telemetry, aerodynamic normalization and power unit stabilization techniques were applied, allowing the simulated data to better reflect the operational characteristics and performance trends observed in modern Formula One racing.

Machine Learning models were trained on the generated telemetry dataset to perform both regression and classification tasks. Regression models were used to predict performance-related variables such as lap times and speed trends, while classification models were employed to evaluate driver performance characteristics and telemetry-based behavioral patterns. The outputs of these models were compared with driver performance trends derived from Formula One datasets to assess how effectively the synthetic telemetry reproduced realistic racing conditions and to validate the accuracy of the developed models.

The project also includes a comparative analysis of dry and wet track conditions, examining the effects of changing surface grip on braking behavior, vehicle speed profiles, and overall performance. Through this comparison, the study highlights how environmental conditions influence telemetry signatures and race outcomes. By integrating telemetry simulation, data visualization, machine learning, aerodynamic normalization, power unit stabilization, and track-condition analysis, this project demonstrates the application of data-driven techniques for performance evaluation and predictive analytics within the context of Formula One motorsport engineering.
